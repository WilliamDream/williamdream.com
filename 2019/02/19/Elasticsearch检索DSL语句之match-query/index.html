Elasticsearch检索DSL语句之match query

Posted on 2019-02-19

Elasticsearch检索DSL语句之match query

1. filter过滤器

1.1 query与filter比较

查询DSL分为两种：一种是query DSL，一种是filter DSL。两种不同的上下文环境目的不同。

query filter
relevance boolean yes/no
full text exact values
not cached cached
slower faster

query：不仅要进行full text search，还要进行相关性评分，更倾向检索文档的匹配程度。

filter：主要进行精确查询过滤，对检索文档是否匹配。

触发filter上下文有三种条件：

  • 在bool查询中指定filter参数或者must_not参数；

  • 在 constant_score查询中指定filter参数；

  • filter aggregation，即filter聚合查询；

2. match all query

2.1 match_all
GET /library/books/_search
{
    "query": {
        "match_all": {}
    }
}
#设置评分 boost
GET /library/books/_search
{
    "query": {
        "match_all": { "boost" : 1.0 }
    }
}

3. Full text queries

3.1 match query
GET /library/books/_search
{
    "query": {
        "match" : {
            "title" : {
                "query" : "java"
            }
        }
    }
}

#使用模糊匹配，"fuzziness": "AUTO"
GET /library/books/_search
{
    "query": {
        "match" : {
            "title" : {
                "query" : "javas",
                "fuzziness": "AUTO"
            }
        }
    }
}
3.2 match_phrase短语查询
#match_phrase短语查询
{
    "query": {
        "match" : {
            "title" : {
              "query": "Elastic search"
            } 
        }
    }
}

#slop参数告诉match_phrase查询词条之间相隔多远时仍然将文档视为匹配。一个词为一个slop，默认slop为0
GET /library/books/_search
{
    "query": {
        "match_phrase" : {
            "title" : {
              "query": "Elastic search java",
              "slop":3

            }

        }
    }
}
3.3 match_phrase_prefix前缀匹配查询
GET /library/books/_search
{
    "query": {
        "match_phrase_prefix" : {
            "title" : "java高"
        }
    }
}

GET /library/books/_search
{
    "query": {
        "match_phrase_prefix" : {
            "title" : {
              "query": "Elastic search java",
              "slop":3,
              "max_expansions" : 50
            } 
        }
    }
}
3.4 muti match query 多字段匹配查询

multi_match 匹配查询多个字段

GET /library/books/_search
{
  "query": {
    "multi_match" : {
      "query":    "java", 
      "fields": [ "title", "type" ] 
    }
  }
}
#支持正则匹配多个字段
GET /library/books/_search
{
  "query": {
    "multi_match" : {
      "query":    "java", 
      "fields": [ "t*", "author" ] 
    }
  }
}

multi_match查询内部执行方式取决于type参数

type可以选择有：best_fields、most_fields、cross_fields、phrase、phrase_prefix。

GET /library/books/_search
{
  "query": {
    "multi_match" : {
      "query":      "java",
      "type":       "most_fields",
      "fields": [  "type","title" ] 
    }
  }
}

4. query string

4.1 query string
GET /library/books/_search
{
    "query": {
        "query_string" : {
            "default_field" : "title",
            "query" : "(java) OR (mysql)" 
        }
    }
}

#mutli fields
GET /library/books/_search
{
    "query": {
        "query_string" : {
            "fields" : ["title", "type"],
            "query" : "java"
        }
    }
}
